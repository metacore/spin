SPIN Device Drivers

The device extensions are derived from supertypes in devcore. The specific devices are
  • console
  • cam - TC Common Access Methods for SCSI disks and tapes
  • fa - TC fore ATM interface
  • fb - TC frame buffer
  • fore - PCI fore ATM interface
  • ln - TC lance ethernet interface
  • lo - loopback network interface
  • tal - AN1 interface
  • tga - PCI frame buffer
  • tt - T3 interface
  • tu - PCI tulip ethernet interface
  • ws - workstation interface for X
