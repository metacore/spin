How to Use the SPIN
Source Documentation

There is a complete set of user documentation included with the SPIN source tree. These files were initially created for our use at the University of Washington and therefore won't be entirely applicable to you're particular situation. For example some of the links may not work because they reference files that are specific to the UW environment; don't worry about this. This documentation should prove valuable in your use of SPIN.

Please note that some of the documents provided with the SPIN source tree are superseded by the iX86 specific documents located in spin/x86_docs/.

There is also a set of technical papers in postscript format provided with this distribution.

If you have questions, please send mail.
