SPIN Kernel

The SPIN kernel consists of the core SPIN services, some initial extensions and startup code, the Modula-3 runtime, and a bottom layer of code for low-level control of the target platform.

Directories

spincore
The core SPIN services
start
Startup code and boot-time extension configuration.
m3core_sa
Core Modula-3 runtime support.
libm3_sa
Useful interfaces.
sal
The standalone-layer. Borrowed code to operate the target hardware.
