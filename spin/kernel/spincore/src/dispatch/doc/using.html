Interface and usage

The dispatcher provides a set of operations that can be grouped according to the feature they support. Each page shows a piece of the dispatcher interface, instructions about its usage, and some examples.

  • Simple installation and uninstallation of handlers
  • Handler set manipulation
  • Default handler and result handling
  • Asynchrony
  • Authorization
  • Imposed guards
